# Χάρτης Πορείας - Ascoos OS

Το Ascoos OS είναι ένας PHP πυρήνας **Web 5.0** και **WebAI** υπό ανάπτυξη.

---

## Στόχοι / Φιλοσοφία

> Το Ascoos OS στοχεύει να αποτελέσει έναν ενιαίο PHP Web 5.0 / WebAI πυρήνα με έμφαση στην αυτοματοποίηση, την τεχνητή νοημοσύνη και την ενοποίηση φυσικής γλώσσας.

Δείτε πώς το Ascoos OS υλοποιεί το Web 5.0 στο [Web 5.0 και Ascoos OS](https://os.ascoos.com/docs/articles/ascoos-meets-web5-el.html) και το [WebAI](https://github.com/ascoos/WebAI/blob/main/README-EL.md).

---

---

## 2025

- ✔️ Δημιουργία/αναβάθμιση ~280 κλάσεων (AI, NLP, δικτύωση, Macro Engine, IoT).
- ✔️ Δημιουργία και βελτίωση τεκμηρίωσης, παραδειγμάτων και μελετών περιπτώσεων.
- ✔️ Εσωτερικές εκδόσεις για αποσφαλμάτωση.

## 2026

- Ολοκλήρωση ~800 κλάσεων συνολικά.
- Επέκταση προηγμένων AI λειτουργιών (νευρωνικά δίκτυα, NLP).
- Επέκταση IoT υποστήριξης.
- Επέκταση Ascoos Macro Engine
- ✔️ Κυκλοφορία **JML** (JSON-Inspired Markup Language).
- Κυκλοφορία online **JML Studio**.
- Κυκλοφορία του Ascoos Documentation Builder (**DoBu**).
- Κυκλοφορία του **BootLib** UI Framework.
- Κυκλοφορία του **jAscoos** Javascript Framework.
- Κυκλοφορία **Ascoos OS Helper** VSCode extension για προγραμματιστές.
- Δημιουργία επίσημης ιστοσελίδας τεκμηρίωσης.

## 2027

- Ολοκλήρωση ~1500 κλάσεων συνολικά.
- Ολοκλήρωση συνοδευτικών εφαρμογών.
- Ολοκλήρωση τεχνολογίας **WIC** (Web Image Cryptography).
- Ολοκλήρωση και εκκίνηση όλων των επίσημων ιστοσελίδων.
- Ολοκλήρωση τεχνολογίας **CiC**. [Δες στο Γλωσσάρι](https://os.ascoos.com/docs/api/el/glossary.html#διερμηνεία--ενοποίηση)
- Δημόσια Εμπορική έκδοση Beta, υπό **AGL** (Ascoos General License).
- Πρώτες επίσημες client webpages.

## 2028

- Ανάπτυξη 500+ εγγενών κλάσεων τεχνητής νοημοσύνης, συμβολικά μαθηματικά.
- Εμπορική έκδοση υπό **AGL** (Ascoos General License).
- Ολοκλήρωση **Ascoos Framework** και **Ascoos CMS Oxyzen**.

## 2029 - 2032

- Ολοκλήρωση τεχνολογίας **LiL** (Language in Language). [Δες στο Γλωσσάρι](https://os.ascoos.com/docs/api/el/glossary.html#διερμηνεία--ενοποίηση).
- Πλήρης μεταγλωττιστής φυσικής γλώσσας → DSL, φωνητικές εντολές σε ελληνική και άλλες γλώσσες.

## 2033 - 2040

- Δίκτυο WebAI Mesh – αμφίδρομα νευρωνικά δίκτυα μεταξύ συσκευών.
