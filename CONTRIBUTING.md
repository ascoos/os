# Contributing to Ascoos OS

Thank you for your interest in Ascoos OS, a Web 5.0 Kernel. Due to the source code being encrypted with IonCube, contributions are limited to documentation and examples.

## How to Contribute
- **Feedback**: Report issues or suggestions on the [Bug Tracker](https://issues.ascoos.com).
- **Documentation**: Submit improvements or translations into other languages.
- **Examples**: Submit your own code examples.

## Guidelines
- Use the namespace ***`"ASCOOS\OS\Libs\[***]"`*** for your own **Ascoos OS** classes.
- Respect the Ascoos General License (AGL).

## Contact
Email: [support@ascoos.com](mailto:support@ascoos.com) | [Bug Tracker](https://issues.ascoos.com).
