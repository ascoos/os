# Συνεισφορά στο Ascoos OS

Σας ευχαριστούμε για το ενδιαφέρον σας για το Ascoos OS, έναν πυρήνα Web 5.0. Λόγω κρυπτογράφησης του πηγαίου κώδικα με IonCube, οι συνεισφορές περιορίζονται σε τεκμηρίωση και παραδείγματα.

## Πώς να Συνεισφέρετε
- **Σχόλια**: Αναφέρετε προβλήματα ή προτάσεις στον [Bug Tracker](https://issues.ascoos.com).
- **Τεκμηρίωση**: Υποβάλετε βελτιώσεις ή μεταφράσεις σε άλλες γλώσσες.
- **Παραδείγματα**: Υποβάλετε τον δικό σας κώδικα με παραδείγματα.

## Οδηγίες
- Χρησιμοποιήστε το namespace ***`"ASCOOS\OS\Libs\[***]"`*** για τις δικές σας **Ascoos OS** κλάσεις.
- Σεβαστείτε την Ascoos General License (AGL).
